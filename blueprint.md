# Turizm.ge - Georgia Tourism Portal

## Overview

A minimal, visually appealing tourism portal for Georgia, named "turizm.ge". This application will showcase the beauty and attractions of Georgia, providing a starting point for potential tourists.

## Design and Style

- **Layout:** A modern, single-page layout with a clear visual hierarchy.
- **Color Palette:** Inspired by the Georgian flag and landscape - using reds, whites, and earthy tones.
- **Typography:** Using the "Geist" font, as it is already part of the project, with varied sizes for headings and text to create emphasis.
- **Imagery:** High-quality, evocative images of Georgian landscapes, cities, and culture will be used. Initially, these will be placeholder images.

## Features

- **Header:** A simple header with the site name/logo.
- **Hero Section:** A full-width hero image with a catchy tagline.
- **Destinations Section:** A grid showcasing popular tourist destinations in Georgia (e.g., Tbilisi, Batumi, Svaneti).
- **Cuisine Section:** A section dedicated to Georgian food and wine.
- **Activities Section:** Highlighting various activities like hiking, cultural tours, etc.
- **Footer:** A simple footer with social media links and contact information.
